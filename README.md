## 파일 안내

* `utils.R` : 책에 포함된 여러 함수
  * `printlocales()` : 로케일 설정 사항을 보기 좋게 출력
  * `encoding(x) <- "xxx"` : R의 기본 함수인 `Encoding(x) <- "xxx"`는 인코딩을 반영할 수 없을 때에도 아무런 오류를 발생시키지 않는다. `encoding()`의 경우 `Encoding()`과 기능은 동일하지만 만약 적절히 실행이 불가능한 경우에 오류를 발생시킨다.
